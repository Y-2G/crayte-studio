このたび、CRAYTE STUDIOのコーポレートサイトを全面リニューアルいたしました。今回のリニューアルでは、お客様により分かりやすく、使いやすいサイトを目指して多くの改善を行いました。

## 主な変更点

### レスポンシブデザインの刷新

すべてのページをモバイルファーストで設計し直しました。スマートフォン、タブレット、デスクトップのどのデバイスでも最適な表示でご覧いただけます。

### 制作実績紹介ページの拡充

これまでの制作実績をカテゴリ別に整理し、各プロジェクトの詳細情報を充実させました。フィルタリング機能も追加し、お探しの情報により素早くアクセスできるようになっています。

### スタッフ紹介の追加

CRAYTE STUDIOで働くメンバーの紹介ページを新設しました。どのようなチームが皆様のプロジェクトを担当するのか、より身近に感じていただければ幸いです。

### お問い合わせフォームの改善

お問い合わせフォームをよりシンプルに、かつ必要な情報を的確にお伝えいただけるよう改善しました。レスポンスタイムの短縮にも取り組んでおります。

## 技術スタック

- **フロントエンド**: Next.js (App Router) + React
- **スタイリング**: CSS Modules + CSS Custom Properties
- **ホスティング**: Vercel
- **アニメーション**: CSS Animations + Intersection Observer API

より分かりやすく、使いやすいサイトを目指して改善を続けてまいります。ぜひご覧ください。
