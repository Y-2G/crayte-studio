## プロフィール

私は CRAYTE STUDIO でプロジェクトマネージャーとして、複数のプロジェクトを同時に管理し、チーム全体が円滑に動くための土台をつくっています。大阪で生まれ育ち、関西大学社会学部を卒業後、SI 企業でシステムエンジニアとして 3 年間勤務しました。その後 Web 業界に転身し、大型プロジェクトの PM を経験してから CRAYTE STUDIO に参画しました。

私のモットーは「段取り八分、仕事二分」です。仕事の成否は、実行よりも準備段階でほぼ決まると考えています。タスクを細かく分解し、リスクを事前に洗い出し、スケジュールに余裕を持たせる。この準備があるからこそ、チームは安心して創造的な仕事に集中できます。CRAYTE STUDIO では、デザイナーやエンジニアが最高のパフォーマンスを発揮できるよう、裏方として支える役割を大切にしています。

## キャリア

SI 企業では、金融機関や自治体向けのシステム開発に携わりました。厳格なスケジュール管理とリスクマネジメントが求められる環境で、プロジェクト管理の基礎を徹底的に学びました。ただ、次第に「もっとユーザーに近い、創造的なプロジェクトに関わりたい」という思いが強くなり、Web 業界への転職を決意しました。

Web 系企業では、EC サイトの大規模リニューアルや、コーポレートサイトの多言語展開など、複雑なプロジェクトを数多く担当しました。クライアント、デザイナー、エンジニア、外部パートナーなど、多様なステークホルダーを調整し、期限内に高品質な成果を出す経験を積みました。CRAYTE STUDIO には、よりクリエイティブな環境で PM としてのスキルを磨きたいと思い、参画しました。

## 仕事について

私の仕事は、プロジェクトの全体像を把握し、タスクを適切に分解して、各メンバーにアサインすることから始まります。その際、各メンバーのスキルや稼働状況、得意分野を考慮し、無理のない配分を心がけます。また、タスクには必ず期限とマイルストーンを設定し、進捗を可視化します。私が愛用しているのは、Notion や Asana といったプロジェクト管理ツールです。これらを駆使して、誰が何をいつまでにやるかを明確にします。

リスク管理も重要な仕事です。プロジェクトには必ず不確実性が伴います。クライアントの要件変更、技術的な障壁、メンバーの急な欠員など、予期せぬ事態は必ず起こります。だからこそ、事前にリスクをリストアップし、発生した際の対応策を準備しておくことが大切です。問題が起きてから慌てるのではなく、冷静に対処できる体制を整えておく。それが私の役割です。

## こだわり

私がこだわっているのは、「透明性」と「公平性」です。プロジェクトの状況は全メンバーに共有し、誰もが同じ情報を持てるようにします。情報の偏りは誤解や不信感を生み、チームのパフォーマンスを下げます。また、タスク配分においても公平性を重視します。特定の人に負担が偏らないよう、常にバランスを見ながら調整します。

もう一つは、「余裕の確保」です。スケジュールはギリギリで組まず、必ずバッファを設けます。クリエイティブな仕事は、時間に追われると質が下がります。余裕があるからこそ、デザイナーは納得いくまでブラッシュアップでき、エンジニアは丁寧にコードを書けます。無理のないスケジュールを組むことが、結果的に高品質な成果につながると信じています。

## プライベート

私は手帳術が趣味です。仕事でもプライベートでも、紙の手帳に予定やタスク、思考を書き込みます。デジタルツールも使いますが、手書きの方が記憶に定着しやすく、頭の整理がしやすいと感じています。毎年新しい手帳を選ぶのも楽しみの一つで、文房具店で何時間も悩むこともあります。

整理整頓も好きで、自宅は常にすっきりと片付いています。物の定位置を決め、使ったら必ず元に戻す。この習慣が、仕事でのタスク管理にも通じていると思います。読書も好きで、特にミステリー小説をよく読みます。伏線の張り方や、緻密な構成に触れることで、プロジェクト設計のヒントを得ることもあります。物語の構造とプロジェクトの構造には、意外と共通点があるんです。
